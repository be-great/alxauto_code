## Runs :-

### 1- To create the executable file  
    $ make
### 2- To run test cases
    $ make test
### 3- To clean created files
    $ make clean
### 4- create the def package and save it on the pack folder
    $ make package
## Requirments
### 1- Ctags installed
    $ sudo apt-get update
    $ sudo apt-get install exuberant-ctags

### 2- Function be in betty style
    $ betty *.c
